# Airbnb NYC - SQL Analysis

## Overview
SQL analysis on Airbnb NYC Open Data Using SQL Server (SSMS)

## Objective
- Identify most popular neighbourhoods
- Analyze pricing by room type and location
- Understand host distribution and cancellation policies

## Tools Used
- SQL Server (SSMS)
- SQLite Online
